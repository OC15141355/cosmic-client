## Recommendations for TwinleafStory Scroll Economy

### 1. Scroll Availability Tiers

**NPC Shop Scrolls (always available, meso sink):**
- All 100% scrolls (low stat bonus, safe option)
- Basic cure items (Antidote, Eyedrop, Tonic, Holy Water, All Cure)
- Return Scroll - Nearest Town
- Town-specific return scrolls

**Monster Drops (regular scrolls):**
- 60% scrolls -- common drops from level-appropriate monsters
- 70% Dark scrolls -- uncommon drops from mid-level monsters
- 30% Dark scrolls -- rare drops from high-level monsters

**Boss-Exclusive Drops:**
- 10% scrolls (high stat) -- Zakum, Horntail, Pink Bean
- Clean Slate Scrolls -- Horntail, Pink Bean
- Chaos Scroll 60% -- Horntail only
- White Scroll -- Pink Bean only (or very rare Horntail)
- Skill Books -- specific boss drops (Horntail, etc.)

**Event/Special Scrolls (controlled release):**
- 4yrAnniv/6thAnniv scrolls -- seasonal events only
- King Pepe scrolls -- PQ rewards
- 50% scrolls -- anniversary events or special PQs
- 65%/15% scrolls -- Balrog, special content
- Balrog's special shoe scrolls -- Balrog boss drop

### 2. Post-v83 Slot Scroll Distribution

**Belt Scrolls (2041300-2041319):**
- 100% from NPC shop
- 60% drops from Von Leon or Empress monsters
- 10% from Von Leon boss
- Dark 70%/30% from Empress boss

**Ring Scrolls (2041100-2041119):**
- 100% from NPC shop or ring quest rewards
- 60% from Horntail, Zakum
- 10% from Horntail
- Dark versions from boss drops

**Pendant/Accessory Scrolls (2049200-2049211):**
- Dark Scrolls for Accessory are universal -- use for HT pendant
- 70% from Horntail prequest bosses
- 30% from Horntail itself

**Shoulder Scrolls:**
- NOT present in repack data -- **need to be created custom**
- Recommend adding IDs in 2041400-2041419 range following belt/ring pattern
- Or use the Dark Scrolls for Accessory as a universal solution

### 3. Scroll Economy Balance

**Remove/Disable:**
- Potential scrolls (2049xxx Potential-related) -- no Potential system in v83
- GM scrolls (2044818, 2044911) -- not for players
- MISSING NAME items (2100141-2100166, 2280020-2280025) -- broken entries

**Prices (NPC 100% scrolls):**
- Armor scrolls: 5,000-20,000 meso
- Weapon scrolls: 30,000-50,000 meso
- Accessory scrolls: 10,000-30,000 meso

**Drop Rate Suggestions:**
- 60% scrolls: 1/200 drop rate from level-appropriate mobs
- Dark 70% scrolls: 1/500 from level 50-80 mobs
- Dark 30% scrolls: 1/1000 from level 80+ mobs
- 10% scrolls: 1/50 from bosses (Zakum/HT/PB)
- Clean Slate 5%: 1/20 from Horntail
- Chaos Scroll: 1/10 from Horntail
- White Scroll: 1/50 from Horntail, 1/20 from Pink Bean

### 4. Missing Content to Create

1. **Shoulder Pad Scrolls** -- Need to add for Von Leon/Empress shoulder pads
2. **Pendant-specific scrolls** -- Currently only generic Accessory Dark Scrolls exist; consider adding dedicated pendant scrolls
3. **Scroll for Shoes for ATT** (2040758-760) -- Already exists, good for Pirate/Thief builds
4. **Verify all Knuckle/Gun scrolls** have proper stat values since they were added later in v83

### 5. Economy Integrity

- **White Scroll + 10% scroll combo** should be the endgame upgrade path
- **Clean Slate Scrolls** should be rare enough to maintain value but available enough to be usable
- **Chaos Scrolls** are a gamble mechanic -- keep rare but not impossibly so
- **No Potential system** means scrolling IS the primary equipment progression -- balance accordingly
- **Consider scroll of shop availability at Mu Lung Dojo** as a points reward (100%/60% scrolls)

### 6. Mastery Book Distribution

All 140 mastery books are present. Recommend:
- Common mastery books from Monster Book drops and quests
- Rare mastery books from bosses (Zakum, Horntail)
- Ensure every 4th job class has accessible mastery book sources
- Aran mastery books from Aran-specific content (Rien, Maha)
