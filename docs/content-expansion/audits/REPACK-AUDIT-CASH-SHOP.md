## Implementation Notes

1. **Deduplication needed**: The 5010084-5010141 range contains ~40 items that are timed sale/event copies of ~8 base effects. Only stock the base version of each in the shop.

2. **Location-specific coupons**: For hair/face/lens coupons, consider whether TwinleafStory will honor all town locations (Henesys, Orbis, Ludi, Showa, Amoria, Night Market, Mu Lung, Ariant, NLC, CBD) or consolidate to fewer locations. If consolidating, only stock coupons for active towns.

3. **REG vs VIP pricing**: REG (random result) coupons should be significantly cheaper than VIP (player's choice) coupons. Suggested: REG = 500k mesos, VIP = 2M mesos.

4. **Pet food variety**: Many pet foods are species-specific. Stock based on which pets are actually obtainable in the game.

5. **Hired Merchant skins**: The paired IDs (5030000/5030001, etc.) appear to be duration variants (24hr/permanent or similar). Verify in Item.wz data before stocking.

6. **Season Challenger Emblems** (5010168-5010172): These are achievement-based effects. Should be earned through PvP/ranking systems, not sold.

7. **Item.wz Cash directory**: The 48 files in `Item.wz/Cash/` (0501-0599 series) contain the actual item properties (price, duration, slot counts, etc.) which should be cross-referenced when setting meso prices.
